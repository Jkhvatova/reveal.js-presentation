React Native

What is React Native?

React Native is an open-source JavaScript framework

designed for building cross-platform apps

developed by Facebook in 2015

React Native uses JavaScript to compile the app’s user interface, but using native-OS views

For more complex features, it allows code implementation in OS-native languages

Why cross-platform development?


Cross-platform development allows:

- to create software that is compatible with multiple platforms or devices using a single codebase


- to build apps using developers preferred frameworks

React vs React Native

Both frameworks use a similar syntax and development mindset, but they serve an entirely different purpose:

- React is used for web development.

- React Native is used for mobile development.

Pros of React Native

  • speed of development
  • one codebase for two platforms: savings in time and money
  • easy to switch
  • Hot Reloading and debugging
  • perfomance
  • growing developer community
  • constantly evolving

Cons of React Native

  • a relatively young technology (but improving with time)
  • the need for native mobile developers
  • React Native does not go well with complex designs & interactions
  • The need for native mobile developers
  • reliance on Facebook
  • near-native performance is still lesser than native

How popular React Native is?

According to 2022 Stack Overflow Developer Survey, React Native is one of the most popular frameworks.


Numerous big players are actively using React Native in their products, or have just decided to switch.
